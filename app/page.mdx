# Installation

## Install the package

Add this to `.npmrc`:

```bash
//npm.pkg.github.com/:_authToken=YOUR_GITHUB_PERSONAL_ACCESS_TOKEN
@enefit:registry=https://npm.pkg.github.com
```

Then, install the package in your project:

```bash
npm install @enefit/eds
```

## Set up the styles

Then import the styles in your main layout file:

```ts
import '@enefit/eds/styles.css';
```

## Use the components

You can import the components you need in your files:

```ts
import { Text } from '@enefit/eds'

export default function Page() {
    return <Text>Hello world!</Text>
}
```
