# Installation

## Install the package

First, install the package in your project:

```bash
npm install another-design-system
```

## Set up the styles

Then import the styles in your main layout file:

```js
import styles from 'another-design-system/styles.css'
```

## Use the components

You can import the components you need in your files:

```js
import { Text } from 'another-design-system'

export default function Page() {
    return <Text>Hello world!</Text>
}
```
