# Command reference

## Creating a new component

To ease the process of creating components you can run the following command:

```bash
npm run new Something
```

This command creates a `Something` component plus style file, test file, documentation file and updates the component and documentation index files. 

## Making a new release

To make a new npm relase, run the following command:

```bash
npm run release
```

This command runs the tests, builds the library, updates the package version and publishes the package to npm.

⚠️ Make sure you have all the git changes commited before running this command.
