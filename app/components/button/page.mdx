# Button

## Usage

```jsx
import { Button } from 'another-design-system'
```

## Basic example

```jsx
<Button>Hello World</Button>
```

<Button>Hello World</Button>

## Variant

```jsx
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="tertiary">Tertiary</Button>
<Button variant="alert">Alert</Button>
```

<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="tertiary">Tertiary</Button>
<Button variant="alert">Alert</Button>

## Icon buttons

### Icon on left

```jsx
<Button iconLeft={<IconUserSm aria-hidden="true" />}>Self-service</Button>
```

<Button iconLeft={<IconUserSm aria-hidden="true" />}>Self-service</Button>

### Icon on right

<Button iconRight={<IconChevronRightSm aria-hidden="true" />}>Self-service</Button>

## Icon only button

Note that `tooltipLabel` prop adds a visible tooltip  to the button, which is also used as `aria-describedby` for screen readers.

```jsx
<Button isIcon tooltipLabel = "Self-service">
    <IconUserSm aria-hidden="true" />
</Button>
```

<Button isIcon tooltipLabel = "Self-service">
    <IconUserSm aria-hidden="true" />
</Button>


## Full width button

```jsx
<Button isFullWidth>Full width button</Button>
```

<Button isFullWidth>Full width button</Button>

## Button as a link

Element `a` is usable only with regular button variant, not icon button and no tooltip.

```jsx
<Button element="a" href="https://google.com" target="_blank">Link to Google</Button>
```

<Button element="a" href="https://google.com" target="_blank">Link to Google</Button>

## Integration with NextJS Link

```jsx
import Link from 'next/link';

<Link href="/test" legacyBehavior passHref>
    <Button element="a">Submit</Button>
</Link>;
```
