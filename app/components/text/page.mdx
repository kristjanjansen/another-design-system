# Text

## Usage

```jsx
import { Text } from '@enefit/eds'
```

## Basic syntax

```jsx
<Text>Example Text</Text>
```

<Text>Example Text</Text>

## Size

```jsx
<Text size="large">Here is large text</Text>
<Text size="default">Here is default text</Text>
<Text size="small">Here is small text</Text>
<Text size="tiny">Here is tiny text</Text>
```

<Text size="large">Here is large text</Text>
<Text size="default">Here is default text</Text>
<Text size="small">Here is small text</Text>
<Text size="tiny">Here is tiny text</Text>

## Weight

```jsx
<Text weight="400">Here is text with default 400 weight</Text>
<Text weight="600">Here is text with 600 weight</Text>
```

<Text weight="400">Here is text with default 400 weight</Text>
<Text weight="600">Here is text with 600 weight</Text>

## More options

Text support more options for styling:

```jsx
<Text italic>Here is the italic label</Text>
<Text underline>Here is the underline label</Text>
<Text uppercase>Here is the uppercase label</Text>
```

<Text italic>Here is the italic label</Text>
<Text underline>Here is the underline label</Text>
<Text uppercase>Here is the uppercase label</Text>
