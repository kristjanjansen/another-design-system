# Banner

## Import

```jsx
import { Banner, Button, IconWindmillLg } from '@enefit/eds'
```

## Basic example

```jsx
<Banner
    icon={<IconWindmillLg />}
    title="Banner title"
    description="Banner description that provides additional information, instructions or giver a context to the user"
>
    <Button element="a" variant="secondary">Banner button</Button>
</Banner>
```
