# Label

## Import

```jsx
import { Label } from 'another-design-system'
```

## Usage

```jsx
<Label>Here is the label</Label>
```

<Label>Here is the label</Label>

## Size

```jsx
<Label size="large">Here is large label</Label>
<Label size="default">Here is default label</Label>
<Label size="small">Here is small label</Label>
<Label size="tiny">Here is tiny label</Label>
```

<Label size="large">Here is large label</Label>
<Label size="default">Here is default label</Label>
<Label size="small">Here is small label</Label>
<Label size="tiny">Here is tiny label</Label>

## Weight

```jsx
<Label weight="400">Here is label with default 400 weight</Label>
<Label weight="600">Here is label with 600 weight</Label>
```

<Label weight="400">Here is label with default 400 weight</Label>
<Label weight="600">Here is label with 600 weight</Label>


## More options

Labels support more options for styling:

```jsx
<Label italic>Here is the italic label</Label>
<Label underline>Here is the underline label</Label>
<Label uppercase>Here is the uppercase label</Label>
```

<Label italic>Here is the italic label</Label>
<Label underline>Here is the underline label</Label>
<Label uppercase>Here is the uppercase label</Label>
